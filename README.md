# TRAIN — Tradutor de Palavras de Informática

Este projeto foi desenvolvido em **Java** para a disciplina de **Programação Orientada a Objetos**.

O sistema consiste em um tradutor simples de palavras da área de informática, permitindo a tradução entre **inglês, espanhol e português**. As traduções são armazenadas em arquivos **CSV**, e a interface gráfica foi desenvolvida utilizando **Java Swing**.

## Conceitos de Programação Orientada a Objetos
O projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de POO, como:
- Classes e objetos  
- Construtores   
- Abstração  
- Tratamento de exceções  
- Uso de coleções (`ArrayList`)  

## Organização do Projeto
- `dicionario` → Lógica de tradução e manipulação dos idiomas  
- `dados` → Arquivos CSV com as palavras dos idiomas  
- `imagens` → Imagens das bandeiras dos países  
- `TelaDicionario` → Interface gráfica do sistema
